# Portfolio Site

## ■ 概要
Webデザイナー職への応募を想定して制作したポートフォリオサイトです。
デザイン設計からコーディング、アニメーション実装まで一貫して制作しました。

## ■ 使用技術
HTML / CSS / JavaScript  
Photoshop / Illustrator  

## ■ 制作期間
約3週間

## ■ 制作ポイント
・縦2分割カラム構成によるレイアウト設計  
・ロードアニメーションの実装  
・Adobe製品アイコンの自作（Illustrator使用）  
・Worksセクションのモーダル機能実装（JavaScript）

## ■ こだわり
デザインだけでなく、構造設計や動きの演出にも力を入れました。
特にユーザー体験を意識したアニメーション設計と、視認性を考慮したレイアウトバランスを工夫しています。

## ■ 補足（対応状況）
本サイトはPCでの閲覧を前提に制作しており、現時点ではスマートフォン向けのレスポンシブ対応は行っていません。
今後、優先度の高いページから順次レスポンシブ対応を進める予定です。

## ■ 今後の改善点
・レスポンシブ対応（スマートフォン / タブレット）  
・コードのモジュール化・保守性向上  
・アニメーション処理の最適化  
・アクセシビリティへの配慮

## ■ 推奨閲覧環境
PC（Chrome / Edge 最新版）
※本ポートフォリオはPC閲覧を前提として制作しています。
